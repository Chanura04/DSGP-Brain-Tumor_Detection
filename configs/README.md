# CONFIGURATIONS

###  Purpose:
Stores YAML/JSON files with hyperparameters, model parameters, 
dataset paths, training settings.

### Examples:
- baseline.yaml
- cnn_small.yaml
- transformer_finetune.yaml

### Includes:
- dataset paths
- learning rate
- batch size
- epochs
- model architecture settings

### Notes:
- Keeps experiments repeatable and consistent.
- YAML is preferred.
